Classwork 3 - GitHub Classroom


Tips on Completing

  • Ever used Word's equations editor? It's not bad, and accepts syntax akin to Latex's (e.g., you can write \sum to create the large sigma).

  • Want to convert written work to the digital? Go ahead and collaborate using another service, and then simply copy-paste into the answer boxes in the Word doc or with a PDF editor (easier on Macs with Preview than on Windows machines).

  • Neither of the above really doing it for you? Prefer something more physical? Print it out and scan it if that's preferable, though ensure that you have a means of collaborating remotely.

  • Warning: importing the document into Google Docs may cause some equations to lose expressions -- double check that you haven't lost anything to the left of a conditioning bar compared to the PDF if you dare to go the Google Doc route!
